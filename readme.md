# PIN

Create a new Python Project

## Installation

Clone the repository then install the pin package.

```
pip install --user -e .
```

## Use

Go to the folder to build the project and execute
``` 
python -m pin
```

A new project with the structure 
```
project_name/
    project_name/
    config/
    builds/
    setup.py
```

will be created.
